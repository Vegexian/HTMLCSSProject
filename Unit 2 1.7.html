Describing The Correct File-Types For Websites

There are several file-types that are accepted for websites that the website interpret in particular ways to enhance the website itself. These are:
-------------------------------------------------------------------
".AVI – Audio/Video Interleaved:
This is a standard video format supported on the windows platform. They do not stream, however, so you have to download the entire file before you can watch any of it."
-------------------------------------------------------------------
".CSS – Cascading StyleSheet:
Css files are a file-type used to store the styling of HTML pages that the HTML page can call to when displaying the website. This is where the font, font colour, background colours, page layout and many other customisations can be applied."
-------------------------------------------------------------------
".DOC – Microsoft Word DOCument:
Microsoft Word is the most popular word processing software in the world and there aren’t many applications that can open this file-type whilst keeping the format 100% true to the original."
-------------------------------------------------------------------
".EXE – EXEcutable File:
This is a file-type that is most seen when downloading an application or program. These can be downloaded from websites that will have the file stored in a server that the website connects to when you click download."
-------------------------------------------------------------------
".GIF – Graphics Interchange Format:
The most common image format on the internet. This can be used to loop a small video clip or series of images for entertainment purposes."
-------------------------------------------------------------------
".HTML/.HTM – HyperText Markup Language File:
The majority of websites that are created use HTML 5/HTM for the basic webpage layout which are then coupled with CSS pages to help customise them."
-------------------------------------------------------------------
".JPG/.JPEG – Joint Photographic Experts Group File:
Another very common image file-type that is mostly used for photos."
-------------------------------------------------------------------
".MIDI/.MID – Musical Instrument Digital Interface:
MIDIs are digitally sequenced music files made primarily on MIDI keyboards and using software within a recording program to alter the sound of the keyboard to imitate other instruments."
-------------------------------------------------------------------
".MP3 – MPEG Layer 3 Sound File:
MP3 is a sound file-type which is highly compressed, that can be downloaded from websites. They have incredibly high quality despite the low file size after compression."
-------------------------------------------------------------------
".MPEG/ .MPG - Motion Picture Experts Group file:
One of the standards for streaming movies."
-------------------------------------------------------------------
".MOV/ .QT - QuickTime MOVie:
The QuickTime format was designed by Apple and originated on the Mac, but has made the transition to the PC and is hugely popular."
-------------------------------------------------------------------
".PDF - Portable Document Format:
Adobe Acrobat files were invented so that documents could be transferred between computers and platforms and still look the exact same. One of the user-friendliest file-types around."
-------------------------------------------------------------------
".PNG - Portable Network Graphics:
PNGs are a file format designed to be used in place of GIFs. They are usually slightly smaller, and sport advanced features like alpha-channel transparency and 24-bit colour support."
-------------------------------------------------------------------
".RAM - Real Audio Movie:
Real Networks created formats for streaming audio and video, and gave away free players for the formats, before allowing themselves to become so smothered in advertising that everyone with sense decided to stop using their programs."
-------------------------------------------------------------------
".RAR - RAR Archive:
This is a compressed file format similar to the popular .zip format. It sports advanced functions like special multimedia compression and has many benefits over zip files."
-------------------------------------------------------------------
".TIFF - Tagged Image File Format:
For really high quality images, TIFFs are used, but cannot be viewed through a browser."
-------------------------------------------------------------------
".TXT - TeXT File:
The most basic of files, it’s just some text."
-------------------------------------------------------------------
".WAV - WAVe sound File:
A basic, either un- or not very- compressed sound file, usually used for short sound samples."
-------------------------------------------------------------------
".ZIP - ZIPped File:
Zipped files are really groups of other types of files kept together and compressed a bit. Many downloads will consist of zip collections, so be sure to have something to open them with."
